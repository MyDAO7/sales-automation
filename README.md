# sales-automation
Python script to clean  messy excel  data  and generating professional  sales reports with charts and visualizing  tools
# What it does?
it can automatically clean your massive meesy excel data and generates professional report and meaningful visual insights
# Features
Handles missing values, duplicates, branch typos
Creates 5 professional Excel sheets
Adds pie chart (branch distribution)
Adds line chart (monthly trend)
# Output
Cleaned Data sheet
Overall Statistics
Sales by Branch (with pie chart)
Monthly Summary (with line chart)
Top Products by sals

# How to use
1 Place your sales file as `client_sales_data.xlsx`
(your excel file with messy data this name used by me as example)
2 Run: `python code.py`
3 Get `Sales_Report_Final_Formatted.xlsx`
